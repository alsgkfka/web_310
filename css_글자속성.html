글자속성

글자에 관련된 속성은 글꼴, 글자크기, 글자굵기, 기울임, 줄간격, 대소문자, 글자색상 등 주된 7가지를 필수적으로 알고 있어야 함

color를 제외한 6가지는 한줄쓰기 가능
속성명 종류 설명
font-family 컴퓨터에 등록되어 있는 폰트명 / 웹폰트 활용 가능 글꼴을 인용부호에 감싼다
다국적사용자를 대상으로 웹페이지를 제공할 때 serif(명조,) Sans-serif(고딕체) 글꼴 마지막에 작성
font-size 수치에 px등의 단위를 붙여서 지정 기본 16px 100% 기준으로 사용
font-style> italic / normal 글자의 기울기 지정
font-variant small-caps / normal 작은 대문자로 변경
font-weight bold / normal 혹은 숫자 지정
400이 기본값 / 600이상 볼드값		 글자의 두께를 지정
font의 종류에 따라 다르며 일정한 값은 아님
line-height 1은 100%의 값으로 줄간격 없음, 기본값은 1.4(140%) 글자의 높낮이인 행간을 조정

font 라는 속성으로 일렬쓰기 가능
font : (font-weight, font-style, font-variant) (font-size/line-height) (font-family)
세가지 순서대로 작성하되, 글자 크기와 글꼴은 생략 불가
(괄호) 안의 내용은 순서가 바뀔 수 있음 / 실제 작성시 괄호는 생략
font라는 속성이 우선순위이며 나중에 위에 이름값과 같지 않더라도 덮어쓰기 가능

글자관련추가속성

주로 글자나 단어보다는 문장에서 사용하는 속성

  • text-aline
    • 특징 - 왼쪽,오른쪽,가운데,양쪽정렬을 나타냄
    • 주의사항 - 블럭요소에서만 사용 가능
    • justify는 내용안에 줄바꿈이 일어날 만큼 많은 글자양이 있을 경우 보임
  • test-indent
    • 특징 - 문단의 첫 머리 들여쓰기 기능
    • 주의사항 - 블럭요소에서만 사용 가능 / px단위 사용
  • text-decoration
    • underline - 밑줄
    • line-through - 가운데줄(취소선)
    • overline - 윗줄
    • none - 제거
  • letter-spacing - 자간설정 / 단위적용
  • text-transform
    • uppercase - 모두 대문자 변경
    • lowercase - 모두 소문자 변경
    • capitalize - 첫 글자만 대문자
