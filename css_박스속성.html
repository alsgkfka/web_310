박스속성

웹페이지 레이아웃 구성시 가장 중요

  • margin - 여백
  • padding - 영역
  • width, height - 크기

작성법

  • 전체적용 - 50px 하나의 숫자로 네군데 모두 적용
  • 각자적용 - 위쪽 오른쪽 아래쪽 왼쪽 (시계방향) / 모두 단위와 함께 작성, 중간에 쉼표사용 불가
    위아래 좌우 / 두개의 숫자로 첫번째 숫자 - 위아래 같이 적용 , 두번째 숫자 - 좌우 같이 적용
  • 부분적용
    • margin-left: 왼쪽여백
    • margin-right: 오른쪽여백
    • margin-top: 위쪽여백
    • margin-bottom: 아래쪽여백
    • padding-left: 왼쪽영역
    • padding-right: 오른쪽영역
    • padding-top: 위쪽영역
    • padding-bottom: 아래쪽영역

width : 가로크기값
width+(padding+border)*2

height : 세로크기값
height+(padding+border)*2

border

박스의 테두리, 가로세로값에 영향을 줌
색상, 굵기, 모양을 한 번에 모두 표기시 보임

작성법
border: black 2px solid (순서와 관계 없음)
4면 모두 한 번에 나오거나 지정할 수 있음 border-left,righttop,bottom

선의 종류
dashed (긴점선), dottes(잛은점선), double(두줄선), 액자선(groove, ridge, inset, outset), none(없음)

배경

배경색, 배경이미지, 배경반복, 배경위치, 배경고정 등을 설정

  • background-color: 색깔이름, RGB, RGBA 로 표기 / tranparent(투명색)
  • background-image: url("위치주소기재")
  • background-repeat: repeat(가로세로 모두 반복), repeat-x(가로반복), repeat-y(세로반복), no-repeat(반복없음)
  • background-position: 가로위치, 세로위치, 생략시 기본값(left-top)
    숫자로 지정시 왼쪽상단 기준에서 시작됨. 가로세로숫자 위치명기재가능
  • background-attachment: 배경이미지를 고정시킬지 스크롤할지 결정
    scroll, fixed
  • background-size:
    • 100% - 화면을 비율상관없이 꽉 채움
    • contain - 배경이미지를 잘리는 부분 관계없이 이미지우선으로 보임
    • cover - 배경이미지와 관계없이 화면을 꽉채움
