박스속성

웹페이지 레이아웃 구성시 가장 중요

  • margin - 여백
  • padding - 영역
  • width, height - 크기

작성법

  • 전체적용 - 50px 하나의 숫자로 네군데 모두 적용
  • 각자적용 - 위쪽 오른쪽 아래쪽 왼쪽 (시계방향) / 모두 단위와 함께 작성, 중간에 쉼표사용 불가
    위아래 좌우 / 두개의 숫자로 첫번째 숫자 - 위아래 같이 적용 , 두번째 숫자 - 좌우 같이 적용
  • 부분적용
    • margin-left: 왼쪽여백
    • margin-right: 오른쪽여백
    • margin-top: 위쪽여백
    • margin-bottom: 아래쪽여백
    • padding-left: 왼쪽영역
    • padding-right: 오른쪽영역
    • padding-top: 위쪽영역
    • padding-bottom: 아래쪽영역

width : 가로크기값
width+(padding+border)*2

height : 세로크기값
height+(padding+border)*2

border

박스의 테두리, 가로세로값에 영향을 줌
색상, 굵기, 모양을 한 번에 모두 표기시 보임

작성법
border: black 2px solid (순서와 관계 없음)
4면 모두 한 번에 나오거나 지정할 수 있음 border-left,righttop,bottom

선의 종류
dashed (긴점선), dottes(잛은점선), double(두줄선), 액자선(groove, ridge, inset, outset), none(없음)
