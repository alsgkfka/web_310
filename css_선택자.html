선택자의 종류

HTML 파일을 CSS를 통해 꾸밀 수 있고 꾸미기 위해서는 지정된 선택을 해야한다.

작성법

선택자 { 속성명 : 속성값; }

선택자종류 작성방법 설명
전체선택자 * HTML 내부의 모든 태그를 불러서 설정
태그선택자 태그명 HTML 내부의 불려진 모든 태그
아이디선택자 #아이디이름 속성값 ID의 이름을 불러서 특정 ID만 설정
클래스선택자 .클래스명 속성값 class의 이름을 불러서 특정 class만 설정
id 와 class 의 차이점 :
id는 한페이지 한 번만 사용(중복사용안됨)
class는 중복적용 가능 , 한 class에 여러 class명 사용 가능
class명 한 칸 띄어쓰기로 여러 class명 부여 가능
중복선택자 , 여러개의 선택자를 나열해서 사용 가능
중복선택자 붙여쓰기(태그.class명) 지정된 태그 중 해당 class이름만 부름
후손선택자 선택자 선택자 부모자 내부에 있는 요소를 모두 부름
자손선택자 부모자>자손자 다이렉트 자손자에 한해서 설정 가능
