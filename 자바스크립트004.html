요소선택

DOM(Document Object Model)의 html 요소를 선택,생성,삭제 등을

직접선택메서드

메서드이름 설명
getElementByID('') HTML요소 중 ID 속성인 요소만 부름
getElementsByClassName('') HTML요소 중 class 속성인 요소를 모두 부름
getElementsByTagName('') HTML요소 중 tag의 요소를 모두 부름
querySelector('') 선택된 요소 중 첫번째 요소만 선택 / 태그를 제외하고 클래스, 아이디를 부를 때는 css와 동일하게 부름
querySelectorAll('') getElementsByTagName과 같은 성질 단, tagname외 다른 중복 요소 및 클래스도 반영가능
